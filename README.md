## Flappy block

Flappy bird based game. My first Unity project.

U can move to left and right tho

Space between pipes decreases with time

Pb score - 56

```
[W]/[Space]/[ArrowUp] - jump
[A] - move to left.
[D] - move to right.
[G] - godmode.
[R] - die.
```
